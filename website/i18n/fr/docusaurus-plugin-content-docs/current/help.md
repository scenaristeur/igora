# AIDE

- Le code source d'Igora est disponible à l'adresse https://github.com/scenaristeur/igora

- Si vous avez besoin d'aide, vous pouvez :
    - [créer un ticket](https://github.com/scenaristeur/igora/issues),
    - utiliser le [channel discord](https://discord.gg/ZZUZwhQnyj) dédié,
    - créer une [discussion github](https://github.com/scenaristeur/igora/discussions)
    - ouvrir une discussion sur le [Forum (EN et FR)](https://forum.chateaudesrobots.fr/)
    - explorer les [Cours](./Cours/introduction)
    - jeter un oeil à la [FAQ](https://github.com/scenaristeur/igora/blob/main/FAQ.md)
